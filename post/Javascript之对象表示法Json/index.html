JSON简介

JSON是一种轻量级的数据交换格式，易于阅读和编写，易于机器解析和生成。

JSON语法

语法规则

对象表示用{}
数组表示用[]
数据用键值对表示，格式为”name”:”key”
数据间用逗号分隔

JSON名称/值对

JSON名称/值对中名称在前，值在后，二者都在双引号中，中间用：隔开。

“myName”:”Kevin”
等价于JavaScript语句：
myName = “Kevin”

JSON值对

JSON值可以为：

数字（整数或浮点数）
字符串（双引号中）
逻辑值（true或false）
数组（方括号中）
对象（花括号中）
null

JSON结构

JSON结构有两种结构，为数组和对象，此外，通过这两种结构的组合可以形成复杂的机构。

1.对象：”object”:{“key”:”value”,”key”:”value”}
2.数组：”array”:[“php”,”javascript”,”python”]
3.组合：{ “combine”:[{“name”:”Kevin”,”age”:”12”},{“name”:”Bob”,”age”:”18”},{“name”:”chorles”:”16”}] }

JOSN操作

清楚JSON结构后，JSON操作非常简单。

1.赋值：var target = {“combine”:[{“name”:”Kevin”,”age”:”16”},{“name”:”Bob”,”age”:”18”}]}
2.访问：tar.combine[0].name
3.修改：target.combine[0].age = “18”
4.转换: JSON转换为字符串：var targetToStr = JSON.stringify(target)
字符串转换为JSON：var targetToObj = JSON.parse(target)
